# ChatGPT PRepo

ChatGPT PRepo is an open-source Chrome extension that enhances your experience with OpenAI's ChatGPT. It provides a repository of prompts that you can use to interact with ChatGPT in various ways.

## Features

- **Predefined Prompts**: Choose from a variety of predefined prompts to interact with ChatGPT in different modes such as a search engine, a chess-playing bot, a code execution tool, and more.
- **Custom Prompts**: Create your own custom prompts and save them for future use.
- **Filter and Sort**: Easily find the prompt you're looking for with our filter and sort features.
- **Easy to Use**: Simply click on a prompt to use it in your chat with ChatGPT.

## Installation

1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable Developer mode by ticking the checkbox in the upper-right corner.
4. Click on the "Load unpacked" button and select the directory where you cloned or downloaded this repository.

## Usage

After installing the extension, click on the extension icon while you're on the ChatGPT page. A popup will appear with a list of predefined prompts. Click on a prompt to use it in your chat. You can also create your own custom prompts by clicking on the "Custom Prompt" button.
